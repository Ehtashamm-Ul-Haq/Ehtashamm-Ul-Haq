<summary>🔧 Permanent fix: self-host your stats card (2 minutes, recommended)</summary>

The public stats-card service is overloaded and frequently down. To get a stats card that *always* works:

1. Go to https://github.com/anuraghazra/github-readme-stats
2. Click the **"Deploy"** button in the README (deploys your own free copy on Vercel)
3. In your new Vercel project, add an environment variable `PAT_1` with a GitHub Personal Access Token (Settings → Developer Settings → Personal Access Tokens → generate one with `repo` scope)
4. Use your new Vercel URL instead of the public one, e.g.:
   `https://YOUR-PROJECT-NAME.vercel.app/api?username=Ehtashamm-Ul-Haq&show_icons=true&theme=dark`

This removes you from the shared rate limit entirely, so the card loads instantly and reliably.
